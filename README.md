# Patterns for dummies
________________________________________________________


### Description

This is a symfony basic proyect with some basic examples of desing patterns for programming.
This project includes the following patterns:

- Creational Patterns:
    * Factory Method.
    * Abstract Factory.
    * Builder.
    * Prototype.
    * Singleton.

- Structural Patterns:
    * Adapter.
    * Bridge.
    * Composite.
    * Decorator.
    * Facade.
    * Flyweight.
    * Proxy.

- Behavior Patterns:
    * Chain of Responsability.
    * Iterator.
    * Mediator.
    * Memento.
    * Observer.
    * State.
    * Strategy.
    * Template Method.
    * Visitor.

##### Requirements

PHP 8.1
SYMFONY 6.1
COMPOSER 2.1
SYMFONY CONSOLE
MARIADB or MYSQL( compatible with array types )
